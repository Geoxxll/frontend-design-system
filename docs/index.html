Tabs

Tab container with sliding indicator animation. Fires claude-tab-change on switch.

This is the Overview tab content. Tabs manage child <claude-tab> elements and render a tab bar from their labels.

Install the Claude Design System via npm:

npm install claude-design-system

Import the library and use any component:

<script type="module" src="claude-design-system.es.js"></script> <claude-button variant="primary">Click me</claude-button>
ComponentPropTypeDescription
